OVERVIEW

The project was to create a friendly interface for the DSI départment so that the employes got a “wouahou” effet when arriving on the platform. The brief was to create a platform allowing users to request assistance concerning their materiels and softwares, to access a knowledge base and a services catalogue. I’ve run interview, build a userflow and make wireframes. I’ve run usability test in remote and create a UI atomic design. As I collaborate with Servicenow IT employes, I iterate and adapt the platform to the software capabilities.
